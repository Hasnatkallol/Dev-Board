Question-1: What are the different ways to select an element in the DOM?

In js there are 6 ways to select an element in dom :

  • 1. let idName = document.getElementById('idName');
  • 2. let className = document.getElementsByClassName('className');
  • 3. let tagName = document.getElementsByTagName('tagName');
  • 4. let name = document.getElementsByName('name');
  • 5. let allClass = document.querySelectorAll('.allClass');
  • 6. let oneClass = document.querySelector('.oneClass');

Question-5: How do you create, add, and remove elements using JavaScript?

For add an Element we use appendChild() example : let container = document.getElementById('container'); let div = document.createElement('div') container.appendChild(div);

For remove an Element we use removeChild() example : let container = document.getElementById('container'); let div = document.createElement('div') container.appendChild(div); container.removeChild(div)
